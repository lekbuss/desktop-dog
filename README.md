# Desktop Dog

> 一只住在你桌面角落里的柴犬。

---

## 这是什么

这不是普通的桌面挂件，也不是 AI 助手。  
它是一只陪你工作的小狗。

它会在屏幕角落里走来走去，偶尔打个哈欠，困了会趴下来睡觉。  
你不理它太久，它会自己冒出来说一句话。  
你喂它吃东西，它会摇尾巴。  
你主动和它聊天，它会认真回答你。

慢慢的，它会记住你的习惯，开始说出更懂你的话。

---

## 设计理念

### 首先是陪伴，其次才是工具

这只小狗有两个属性，但顺序很重要。

它首先是一只有性格、有情绪的柴犬。  
它的工具能力——提醒喝水、记录事项、提醒活动——是它**照顾主人的方式**，而不是它存在的全部意义。

我们不希望它变成一个会动的效率软件。  
我们希望主人打开电脑，看到桌面角落的它，会觉得"它在"。

### 安静、自然、有分寸

这只小狗的产品气质只有六个字：**不烦人，有存在感。**

它不会频繁打扰你，不会一直发通知，不会把你的桌面搞得乱糟糟。  
它只是在那里，活着，偶尔动一动，偶尔说一句话。

就像真的养了一只狗。

### 三层说话系统

小狗的说话方式分三层，目的是让它**像同一只狗**，而不是时而像客服、时而像玩具：

**本地反应层** — 高频小动作，汪、呜、摇尾巴、打哈欠，不联网，瞬间响应。  
**模板人格层** — 根据当前状态生成短句，「有点饿了」「今天想陪你工作」「你回来啦」，有人格但轻量。  
**LLM 深度层** — 只在真正需要深度的时候调用，主人主动聊天、让它记事、特殊时刻。LLM 是小狗的大脑，不是它的嘴巴。

### 出生仪式

第一次启动不是软件初始化，是一场**认识仪式**。

主人先做简单自我介绍，小狗再介绍自己，然后它"出生"，落到桌面角落。  
这一段有游戏感和仪式感，要让主人觉得不是在配置软件，而是在迎接一只真实的小狗。

性格定调：**傲娇但粘人**。嘴上说不在乎，尾巴已经在摇了。

### 慢慢懂你

小狗理解主人的方式，不是分析、不是贴标签。

它通过一套**倾向值系统**在后台悄悄观察：专注倾向、活动倾向、互动倾向、夜猫子倾向……这些数值随主人的行为慢慢漂移，永远不暴露给主人看。

它的表现方式是自然的：
- 不说「你是专注型用户」
- 而是说「你工作的时候真的很认真，我都不敢乱叫」

小狗是在**形成印象**，而不是在给主人分类。

### 成长，但不刷数值

小狗会随着相处慢慢成长：亲密度、熟悉度、成长阶段，都会变化。

但这套系统是克制的。  
我们不想让主人为了养狗而开始打工。  
成长应该是自然发生的，像真的养宠物，而不是像手游签到。

## 状态与行为

这只柴犬有七种基础状态，每种状态都有对应的动画和行为：

| 状态 | 触发条件 | 表现 |
|------|----------|------|
| 待机 idle | 默认 | 坐着，轻微呼吸，偶尔歪头、扇耳朵、卷尾巴 |
| 散步 walk | 定时或手动 | 缓步移动，四腿错相步态，到达目标后发呆 |
| 奔跑 run | 双击或高精力 | 欢快快跑，舌头伸出，耳朵随风飘 |
| 睡觉 sleep | 夜间 + 精力低 | 趴下闭眼，慢呼吸，冒 zzz |
| 难过 sad | 饥饿或缺水 | 垂头，尾巴低垂轻抖，等待照顾 |
| 被抚摸 pet | 鼠标悬停 | 眯眼，尾巴快速摇，冒爱心 |
| 说话 talk | 主动聊天或自动触发 | 嘴巴动，气泡打字机效果 |

有四个属性值，随时间自然衰减：饱食度、水分、心情、精力。  
属性低到一定程度会影响状态，喂食喂水可以恢复。

## 视觉设计

这是一只纯代码绘制的柴犬，用 SVG 路径描述身体结构，CSS 动画驱动所有动作。

配色是经典柴犬色：橙棕色主毛、奶白色腹部和脸颊、深棕色阴影层。  
脖子上围着一条小围巾。

动画的设计原则是**微动**——幅度小但持续存在，让它在静止时也有生命感：  
呼吸让身体轻微起伏，头部随呼吸有极小的倾斜，眼睛会自然眨眼，尾巴始终在轻轻摆动。

走路用真实的对角步态（右前左后同相），每条腿的 lower 部分有独立的膝盖弯曲动画，爪子落地有轻微的压缩感。

## 工具能力

这些是小狗照顾主人的方式，不是它的主要功能：

- **喝水提醒** — 定时用小狗的方式提醒，不是系统通知
- **活动提醒** — 坐太久了，它会来戳你
- **记事** — 跟它说「记一下 xxx」，之后问它它能告诉你
- **陪伴散步** — 提醒主人短暂离开工位，一起动一动

---

## 关于这个项目

这个项目的目标只有一个：

**让主人在工作的时候，真的觉得桌面角落里有一只自己的狗在陪着自己。**

所以最重要的不是功能有多全，而是：是否自然、是否有灵魂、是否让人愿意一直留着它。
